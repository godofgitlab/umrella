---Об Umbrella---                                                                                                                                                                                                                                      Umbrella — это разработанный сообществом загрузчик скриптов для Dota 2, поддерживающий расширенные функции,  автоматическое комбо, прогнозирование и многое другое. Разработан таким образом, чтобы минимально влиять на производительность и оставаться незамеченным в последних сборках Dota 2.
